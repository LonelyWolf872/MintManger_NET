# MintManger_NET
Search, read and download manga with MintManger.NET
# How to use
* To load all mangas
  * Open app
  * Click on "Home" button
  * Wait while all manga to be loaded
* To search manga
  * Open app
  * Click on "Search" textbox
  * Type name of manga
  * Press "Enter" on your keyboard
  * Wait while list of manga to be loaded
